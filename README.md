# JobTrackr AI

JobTrackr AI is a full-stack job application tracking platform designed for job seekers to manage applications, recruiter outreach, interview preparation, and progress analytics in one place.

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- AWS Lambda
- API Gateway
- DynamoDB
- Cognito

### DevOps
- GitHub
- AWS CDK
- CodePipeline

## Core Features

- User authentication
- Job application tracker
- Recruiter contact tracker
- Interview preparation tracker
- LeetCode progress tracker
- Dashboard analytics
- Resume version tracking

## Project Goals

- Build a production-style full-stack application
- Practice AWS serverless architecture
- Demonstrate REST API design
- Strengthen resume project storytelling
- Prepare for software engineering interviews

## Project Status

In progress.

## Authentication

Authentication is implemented using Amazon Cognito.

API Gateway validates JWT tokens before requests reach Lambda.

This ensures only authenticated users can access protected resources.

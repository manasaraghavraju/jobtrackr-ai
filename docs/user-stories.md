# User Stories

## Authentication

As a user, I want to sign up and log in securely so that my job search data is private.

## Job Application Tracking

As a user, I want to add job applications so that I can track where I applied.

As a user, I want to update the status of each application so that I know whether it is applied, interview, rejected, or offer.

As a user, I want to add follow-up dates so that I do not forget recruiter follow-ups.

## Recruiter Tracking

As a user, I want to save recruiter contact details so that I can manage networking efficiently.

As a user, I want to record when I last contacted a recruiter so that I can follow up professionally.

## Interview Preparation

As a user, I want to track LeetCode problems so that I can monitor my coding progress.

As a user, I want to track resume deep-dive topics so that I can prepare for behavioral and technical interviews.

## Dashboard

As a user, I want to see total applications, interviews, rejections, and follow-ups so that I can measure my job search progress.

## Analytics

As a user, I want to see which companies and roles I am applying to most often so that I can improve my strategy.
